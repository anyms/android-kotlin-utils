# Android Kotlin Utils - A bundle of utility tools

Add it in your root build.gradle at the end of repositories:

```gradle
allprojects {
    repositories {
        ...
        maven { url 'https://jitpack.io' }
    }
}
```

Add the dependency

```gradle
dependencies {
        implementation 'com.github.anyms:android-kotlin-utils:Tag'
}
```
